📡 Getting Started

  1. Add nodes — right-click anywhere on the map → "Add node here". Each node is a transmitter or receiver. Click the ◎ button to centre on your current location first.
  2. Set antenna height — in the sidebar, edit each node's ANT HT (metres above ground). Default is 6 m.
  3. Link nodes — right-click a node marker → "Connect to …" to add a radio link. Links automatically join into paths.
  4. Pick a band — top of the sidebar, choose a radio preset (Meshtastic AU 915 MHz by default) or enter a custom frequency.
  5. Analyse — click ▶ ANALYSE. Terrain elevation is fetched along each link and the line-of-sight + 1st Fresnel zone are drawn below the map.
  6. Coverage layer — set TX power / antenna gain / RX sensitivity (defaults seeded from the preset), tick the COVERAGE box on a node, then click COMPUTE. A terrain-aware coverage polygon is drawn. Use ALL ON / ALL OFF to toggle every node at once.

✓ CLEAR — LOS clear and Fresnel meets preset threshold (40 % for Meshtastic/ham, 60 % for Wi-Fi).  ⚠ MARGINAL — LOS clear but the 1st Fresnel zone is obstructed below the preset threshold (diffraction loss likely).  ✕ BLOCKED — terrain in the way.

Tip: terrain is fetched from AWS Open Terrain Tiles (~30 m global) — no API key, cached in-memory once loaded. Right-click a link on the map to delete it. Use ⇪ SHARE to copy a URL containing your full setup.
